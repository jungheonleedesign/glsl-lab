# 3. 볼류메트릭 연기 — `VolumeSmoke.tsx`

연기를 3D 공간에 두고, 카메라 광선이 공간을 꿰뚫으며 그린다(레이마칭).

## 레이마칭 — 3D를 1D로 환원

모든 것의 토대는 **매개변수 광선**이다.

```
p(t) = ro + rd·t      ro=카메라, rd=픽셀 방향, t=거리
```

3D 기하 문제를 전부 "어떤 t에서?"라는 1D 문제로 바꾼다.
각 픽셀에서 광선을 쏘고 일정 간격으로 **행진(march)**하며 밀도를 모은다.

```
카메라 ●──●──●──●──●──→  스텝마다 그 위치의 3D 밀도를 샘플
```

이는 **연속 적분의 이산 근사**다. 스텝이 크면 빠르지만 밴딩(줄무늬) 오차 →
시작점을 픽셀마다 흔드는 **지터**로 줄무늬를 미세 노이즈로 바꿔 숨긴다.

## 광선-상자 교차 (슬랩)

행진 범위 [진입 tN, 탈출 tF]를 구한다. 상자 = 3쌍 평면(슬랩)의 교집합.

```
교집합 시작 = 가장 늦게 들어간 t (max)
교집합 끝   = 가장 일찍 나간 t (min)
tN > tF 면 광선이 빗나감
```

## 3D 노이즈

2D 쌍선형(4점)을 **삼선형(8점)** 으로 확장. `mix`를 z축으로 한 번 더 감싼다.

## Beer-Lambert — 빛 흡수

```
투과율 = exp(-밀도 × 거리)
```

빛이 한 층 지날 때마다 일정 비율씩 흡수 → 지수 감쇠.
두꺼운 연기일수록 어둡다. 연기·구름·물·피부까지 지배하는 법칙.

## 자기그림자 — 2차 광선

각 행진 점에서 **광원 쪽으로 또 짧게 행진**해 누적 밀도를 재면,
그 점이 빛을 얼마나 받는지(그림자) 나온다 → 입체감.
비용은 메인 N스텝 × 그림자 M스텝 = N×M (곱셈이라 비쌈).

## Henyey-Greenstein 위상함수

빛이 입자에 부딪히면 방향마다 산란량이 다르다(이방성).

```
HG(cosθ, g)   cosθ = dot(보는방향, 빛방향)
  g>0 앞 산란 → 빛 쪽 볼 때 가장자리 광륜(silver lining)
  g=0 등방,  g<0 뒤 산란
```

(이름의 'phase'는 달의 위상에서 온 것. 위상수학과 무관.)

## 다중 산란 근사

단일 산란은 짙은 속이 새까맣게 죽는다. 진짜 구름은 빛이 수십 번 튕겨 속까지 밝다.
이를 **점점 약·깊·흐린 산란들의 합**으로 흉내 낸다.

```
겹마다: 흡수 ↓(더 깊이), 위상 g ↓(더 등방), 기여 ↓(더 약하게)
```

## 역투영 — 마우스 상호작용

2D 마우스를 3D로 되돌린다. 투영(3D→2D)이 깊이 z를 버렸으므로
역투영 결과는 점이 아니라 **광선**. 깊이는 평면과 교차시켜 정한다.

```
unproject = (투영·뷰)⁻¹   (3장의 P·V를 거꾸로)
광선 ∩ 평면 → 3D 점 하나
```
